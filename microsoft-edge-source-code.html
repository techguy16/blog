techguy16 March 16, 2025

You thought Microsoft Edge was proprietary, didn't you. Well, because of Panasonic (of all things), we have the source code for Microsoft Edge.

You read that title right! Panasonic (of all things) shared the Microsoft Edge source code on their website. The page with the downloads is here and the downloads are below:
