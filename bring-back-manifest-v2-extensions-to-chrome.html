Bring back Manifest V2 extensions to Chrome!

techguy16 July 11, 2025

In Chrome 140 (and possibly 139), Manifest V2 extensions are "no longer" supported. This is not the case, as the browser still internally supports these extensions. Through some digging in the Chromium source, we discovered that there is actually a command-line flag to re-enable these extensions. 

When launching Chrome, append the following flag to the executable:

--disable-features=ExtensionManifestV2Disabled,ExtensionManifestV2Unsupported,ExtensionManifestV2DeprecationWarning,ExtensionsManifestV3Only

This flag re-enables Manifest V2 extensions and also hides the deprecation warning in chrome://extensions

This was discovered in the run_py_tests.py file in the Chromium source, which has a test that disables Google's absurd decision. The extension_features.cc file has various "features" relating to disabling Manifest v2, which is where ExtensionManifestV2DeprecationWarning and ExtensionsManifestV3Only were discovered.
